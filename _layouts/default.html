Projects & Artifacts

CS300: Course Planner

This project was created to manage university courses and their prerequisites. I implemented course loading, organization for efficient retrieval, and search/display of prerequisite relationships. A key enhancement was upgrading from a basic Binary Search Tree (BST) to a self-balancing AVL tree, improving lookup times and efficiency for larger datasets. This artifact demonstrates my skills in data structures, algorithmic thinking, and performance-based problem solving. View Project

CS340: Client Server Application

Originally a CRUD-based dashboard for managing shelter animals in a MongoDB database, this project was enhanced with data mining and analytics to provide insights into adoption trends. I implemented aggregation pipelines, added new analytical methods, and visualized adoption patterns using Jupyter Notebook. This artifact illustrates my ability to integrate analytics with software development, working with non-relational databases, and presenting actionable insights. View Project

CS360: Warehouse Application

Developed a Java-based warehouse application with login functionality, item tracking, quantity adjustments, and zero-inventory alerts. The project demonstrates my ability to create full-featured applications, manage data persistence, and ensure system reliability through user-centric design. View Project
